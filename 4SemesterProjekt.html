Kickoff · oprettelse af BMC og begyndelse af planlægning

Mål for dagen

  • Udarbejde BMC
  • Planlægge research inden for backendudvikling, API-design og cybersikkerhed

Lektier

Backendudvikling & API-design – research

  • Azure Monitor API & Azure Resource Graph
    • Hente CPU-, RAM-, disk- og cost-metrics programmatisk
    • Filtrere på inaktive eller overbelastede VM’er direkte i forespørgslen
  • Azure Functions
    • HTTP-trigger → modtage kald og returnere metrics
    • Timer-trigger → automatisk indsamling
  • API-struktur
    • Design endpoints til frontend
    • Returnér både realtidsdata og historiske data

Cybersikkerhed – research

  • Microsoft Authentication (Azure AD)
    • App registration og OAuth2 Authorization Code flow
  • Bearer token validering i Azure Functions
    • Kræv gyldigt token og implementér RBAC
  • Zero Trust-principper
    • Verify explicitly & Least privilege

Anbefalet rækkefølge (POC)

  1. Lav en function/script der henter CPU-data via Azure Monitor API
  2. Opsæt Azure AD app registration og hent token
  3. Kombinér → Azure Function der kun returnerer metrics med gyldigt token
